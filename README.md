# DSA_visualizer

# AlgoViz 🚀

Interactive Data Structures & Algorithms visualizer built with React and TypeScript. Learn DSA through animated visualizations, complexity analysis, quizzes, and interview preparation resources.

## 🌐 Live Demo

https://dsa-explore-viz.lovable.app

## ✨ Features

* Arrays, Stacks, Queues, Linked Lists, BSTs
* Sorting & Searching Visualizations
* Graph Traversals (BFS & DFS)
* Big-O Complexity Cheat Sheet
* Interactive DSA Roadmap
* Interview Preparation Questions
* DSA Quiz with Score Tracking
* Searchable Glossary
* Responsive UI with Animation Controls

## 🛠 Tech Stack

* React 19
* TanStack Start & Router
* TypeScript
* Tailwind CSS v4
* Framer Motion
* shadcn/ui
* Vite
* Bun

## 🚀 Installation

```bash
git clone <repository-url>
cd dsa-viz

bun install
bun run dev
```

Open: `http://localhost:8080`

## 📦 Build

```bash
bun run build
bun run start
```
